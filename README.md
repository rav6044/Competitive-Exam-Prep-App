# 🚀 Competitive Exam Prep App

## 🔥 Features
- Real-time exam system (Socket.io)
- AI-based adaptive questions
- Timer with server sync
- Anti-cheating detection
- Analytics dashboard

## 🛠 Tech Stack
- Frontend: React, Redux Toolkit, Tailwind
- Backend: Node.js, Express, MongoDB
- Realtime: Socket.io

## ⚙️ Setup

### Backend
cd backend
npm install
npm run dev

### Frontend
cd frontend
npm install
npm run dev
